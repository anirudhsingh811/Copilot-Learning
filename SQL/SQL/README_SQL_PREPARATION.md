# SQL Preparation Guide - 2 Week Plan (13 Years Experience)

## Overview
This comprehensive guide covers essential SQL concepts with practical C# demos for interview preparation and skill refreshment.

## 2-Week Study Plan

### Week 1: Core Concepts & Advanced Queries
**Day 1-2: Fundamentals & DDL**
- Database Design & Normalization
- DDL Operations (CREATE, ALTER, DROP)
- Data Types & Constraints

**Day 3-4: DML & Advanced SELECT**
- INSERT, UPDATE, DELETE, MERGE
- Complex JOINs (INNER, LEFT, RIGHT, FULL, CROSS, SELF)
- Subqueries & Correlated Subqueries
- CTEs (Common Table Expressions)

**Day 5-6: Aggregations & Window Functions**
- GROUP BY, HAVING
- Window Functions (ROW_NUMBER, RANK, DENSE_RANK, NTILE)
- LEAD, LAG, FIRST_VALUE, LAST_VALUE
- Partitioning & Ordering

**Day 7: Set Operations & Pivoting**
- UNION, UNION ALL, INTERSECT, EXCEPT
- PIVOT & UNPIVOT
- CROSS APPLY & OUTER APPLY

### Week 2: Performance & Advanced Topics
**Day 8-9: Indexing & Performance**
- Clustered vs Non-Clustered Indexes
- Index Strategies
- Execution Plans
- Query Optimization Techniques

**Day 10-11: Transactions & Concurrency**
- ACID Properties
- Transaction Isolation Levels
- Locking & Deadlocks
- Error Handling (TRY-CATCH)

**Day 12-13: Advanced Features**
- Stored Procedures & Functions
- Triggers (AFTER, INSTEAD OF)
- Views & Materialized Views
- Temp Tables vs Table Variables vs CTEs

**Day 14: Real-World Scenarios**
- Complex Business Logic
- Reporting Queries
- Data Migration Patterns
- Best Practices & Common Interview Questions

## How to Use This Guide

1. Each demo file contains working C# code with SQL queries
2. Run demos sequentially to understand concepts progressively
3. Modify queries to experiment and deepen understanding
4. Focus on execution plans and performance implications
5. Review interview questions at the end of each section

## Setup Instructions

1. Update connection string in demos
2. Ensure SQL Server is running
3. Run demos in order
4. Check output and execution times
